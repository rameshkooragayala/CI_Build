#####################################################
Install kit for XBMC solution for UPC Horizon project
#####################################################


1) Install

	No installation is needed anymore. Normaly at compilation the package
	is detar directly in NDS fs at the correct place.
	And the only installation process needed is in init script now.




2) Run

	To start XBMC run:
	/NDS/xbmc/XBMC_init start

	To stop
	/NDS/xbmc/XBMC_init stop

	To delete the databases and so erase the catalog:
	/NDS/xbmc/XBMC_init reset
